# Resort Off-season LLM Prompt Mission

## 프로젝트 개요
이 프로젝트는 **LLM을 활용해 리조트 비수기 활성화 보고서 개요를 자동 생성**하는 방법을 비교·평가한 실험입니다.

## 문제 상황
- 대상: **한산마리나리조트**
- 객실 수: **31실**
- 보유 요트: **4~6척**
- 비수기 가동률: **30%**

## 목표
- 실무자가 빠르게 보고서 초안을 만들 수 있도록 지원
- 여러 LLM의 응답 품질을 비교해 최적 모델을 선정
- 자사 맥락 반영, 실행 가능성, 환각 억제 수준을 함께 평가

## 실험 설계
- 사용자: **중견 인하우스 마케터**
- 페르소나: **리조트 마케팅 전문 데이터 애널리스트**
- 동일 입력 템플릿과 시스템 프롬프트를 각 모델에 적용
- 평가 기준 6개 축:
  1. 지시 이행도
  2. 자사 맥락 반영도
  3. 인사이트 수준
  4. 환각 억제
  5. 실행 가능성
  6. 톤·형식 적합성

## 현재 결과
| 모델 | 총점(60점 만점) | 평균 |
|---|---:|---:|
| Gemini Pro 3.1 | 54.5 | 9.08 |
| GPT-5.4 | 55.0 | 9.17 |

## 중간 결론
- 두 모델 모두 실무형 개요 생성에 충분히 강한 성능을 보였습니다.
- 현재까지는 **GPT-5.4가 근소 우위**입니다.
- 최종 도입 판단은 점수뿐 아니라 비용, 속도, 조직 접근성까지 함께 고려해야 합니다.
