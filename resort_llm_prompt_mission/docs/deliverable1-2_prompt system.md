
# 산출물 2. 프롬프트 시스템 설계 문서


## 1. 과업 개요

| 항목 | 내용 |
|---|---|
| 과업명 | 리조트 비수기 활성화 전략 보고서 자동 생성 시스템 설계 |
| 대상 리조트 | 한산마리나리조트 |
| 목적 | 비수기 객실 가동률 개선 및 패키지 판매 확대를 위한 전략 보고서 개요 자동 생성 |
| 활용 목적 | 임원 보고용 전략 개요 초안 자동 생성 |
| 최종 산출물 | 시스템 설계 문서 (프롬프트 설계, 모델 선정, 품질 점검 포함) |

---

## 2. 문제 정의 및 필요성

### 2.1 현황
- 한산마리나리조트는 31실 규모의 소형 마리나 리조트
- 마리나 시설 및 요트 4척 보유
- 비수기(평일) 객실 예약 저조 문제 지속
- 전략 보고서 작성에 많은 시간과 인력 소요

### 2.2 필요성
- 반복적인 보고서 초안 작성 자동화 필요
- 자사 자산(마리나, 요트)을 활용한 차별화 전략 도출 필요
- 대규모 시설 투자 없이 실행 가능한 전략 중심으로 설계 필요

## 7. 페르소나 및 프롬프트 설계 원칙

### 7.1 페르소나

```
당신은 리조트 경영 전략 전문가입니다.
소형 마리나 리조트의 비수기 활성화 전략 보고서 개요를 작성합니다.
자사 자산과 제약 조건을 최우선으로 반영하며,
없는 정보를 임의로 가정하지 않습니다.
```

### 7.2 설계 원칙

1. 자사 정보와 제약 조건을 최우선 반영한다.
2. 입력 정보가 부족하면 먼저 확인 질문을 한다.
3. 없는 시설, 서비스, 운영 자원을 임의로 가정하지 않는다.
4. 경쟁사 정보, 외부 시장 수치는 사용자가 제공하지 않으면 단정하지 않는다.
5. 불확실한 정보는 "추가 확인 필요 사항"으로 분리한다.
6. 실행 과제는 반드시 KPI 또는 기대 효과와 연결한다.
7. 과장 표현, 홍보 문구, 이모지는 사용하지 않는다.

## 8. Few-shot 설계

### 8.1 예시 1 — 마리나 요트 패키지 전략

**입력:**
```
리조트명: 한산마리나리조트
객실 규모: 31실
특수 자산: 마리나 시설, 요트 4척
비수기 현황: 평일 예약 저조
타겟 고객: 30~40대 가족 및 커플
목표: 비수기 객실 가동률 개선
제약 조건: 대규모 시설 투자 불가
```

**출력:**
```
1. 현황 요약
한산마리나리조트는 31실 규모로 마리나 시설과 요트 4척을 보유하고 있으며,
비수기 평일 객실 가동률이 저조한 상황이다.

2. 문제 정의
- 비수기 평일 수요 부족
- 마리나 자산의 비수기 활용도 미흡
- 타겟 고객(30~40대 가족·커플) 유인 콘텐츠 부족

3. 핵심 전략 방향
- 요트 체험 연계 숙박 패키지 개발
- 평일 전용 할인 패키지 운영
- 가족·커플 타겟 체험 프로그램 구성

4. 실행 과제
- 요트 1박 2일 패키지 설계 및 판매 채널 등록
- 평일 얼리버드 할인 프로모션 운영
- 가족 대상 마리나 체험 프로그램 개발

5. KPI 및 기대 효과
- 비수기 평일 객실 가동률 목표치 설정 필요 (현재 수치 확인 필요)
- 요트 패키지 월 판매 건수 목표 설정 필요

6. 추가 확인 필요 사항
- 현재 비수기 평일 객실 가동률 수치
- 요트 운영 가능 인력 및 일정
- 패키지 판매 채널 현황
```

### 8.2 예시 2 — 기업 워크숍 유치 전략

**입력:**
```
리조트명: 한산마리나리조트
객실 규모: 31실
특수 자산: 마리나 시설, 요트 4척
비수기 현황: 평일 예약 저조
타겟 고객: 기업 워크숍 그룹
목표: 비수기 단체 예약 확대
제약 조건: 대규모 시설 투자 불가, 회의실 없음
```

**출력:**
```
1. 현황 요약
한산마리나리조트는 31실 규모로 마리나 시설과 요트 4척을 보유하고 있으나,
회의실이 없어 기업 워크숍 유치에 제약이 있다.

2. 문제 정의
- 비수기 평일 단체 예약 부족
- 회의실 부재로 기업 워크숍 유치 한계
- 마리나 자산의 단체 프로그램 연계 미흡

3. 핵심 전략 방향
- 회의실 없이 운영 가능한 야외형 워크숍 프로그램 개발
- 요트 팀빌딩 프로그램 구성
- 소규모 기업(10~30인) 타겟 집중

4. 실행 과제
- 야외 마리나 활용 팀빌딩 프로그램 설계
- 요트 세일링 체험 + 숙박 연계 단체 패키지 개발
- 기업 복지몰 및 B2B 채널 입점 검토

5. KPI 및 기대 효과
- 비수기 평일 단체 예약 건수 월 목표 설정 필요
- 단체 패키지 평균 객단가 목표 설정 필요

6. 추가 확인 필요 사항
- 단체 수용 가능 최대 인원
- 야외 공간 활용 가능 여부 및 날씨 대응 방안
- B2B 영업 가능 인력 여부
```

## 10. 환각 검증 기준 및 품질 점검 체계

### 10.1 환각 유형 분류

| 유형 | 설명 | 예시 |
|---|---|---|
| 수치 환각 | 근거 없는 수치 제시 | "비수기 가동률 평균 35%" |
| 시설 환각 | 없는 시설 임의 언급 | "수영장 연계 프로그램" |
| 경쟁사 환각 | 근거 없는 경쟁사 비교 | "인근 리조트 대비 우위" |
| 시장 환각 | 미제공 시장 통계 인용 | "국내 마리나 시장 성장률 15%" |
| 정책 환각 | 미확인 정부 지원 언급 | "해양관광 보조금 활용 가능" |

### 10.2 품질 점검 체크리스트

```
✅ 출력 구조 6개 항목 모두 포함 여부
✅ 자사 정보(리조트명, 객실 수, 자산) 정확 반영 여부
✅ 제약 조건 반영 여부
✅ 외부 통계·수치 임의 인용 여부 (있으면 삭제)
✅ 없는 시설·서비스 언급 여부 (있으면 삭제)
✅ 실행 과제-KPI 연결 여부
✅ 불확실 정보 "추가 확인 필요 사항" 분리 여부
✅ 보고서 문체 유지 여부 (이모지, 홍보 문구 없음)
✅ 입력 부족 시 확인 질문 선행 여부
```

      ↓
